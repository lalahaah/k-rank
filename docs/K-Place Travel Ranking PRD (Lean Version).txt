🗺️ K-Place: The "Local Selections" Guide PRD (v1.1)| 버전 | 날짜 | 작성자 | 상태 || v1.1 | 2026-01-24 | 넥스트 엔진 | 로컬 데이터 기반 기획 (Source: VisitKorea / 대한민국 구석구석) |1. 제품 비전 (Vision)"Explore Korea like a local, not a tourist."유료 API와 복잡한 크롤링 없이, 한국 관광공사의 공공 데이터를 활용하여 한국인들이 실제로 가장 많이 방문하는 '찐' 여행지 랭킹을 제공한다.2. 핵심 전략 (The Lean Strategy)공공 데이터 활용 (TourAPI): 유료 구글 API 대신 한국관광공사의 TourAPI를 사용하여 여행지 명칭, 주소, 다국어 설명, 고화질 이미지를 무료로 수집.데이터 신뢰도: '대한민국 구석구석'의 조회수(View Count)와 좋아요(Like) 데이터를 기반으로 랭킹 산출.수익 모델 전환: 아마존 대신 트래블 어필리에이트(Klook, Viator, GetYourGuide) 링크를 활용하여 예약 수수료 창출.3. 주요 기능 명세 (Feature Specs)3.1. Local Choice Ranking (여행지 랭킹)Ranking Logic: (TourAPI 조회수 60%) + (SNS 언급량 30%) + (AI 추천 10%).Categorization: * Nature: 산, 바다, 섬.Culture: 사찰, 고궁, 박물관.Modern: 카페 거리, 팝업 스토어 밀집 지역.3.2. AI Cultural GuideContent: 단순 정보가 아닌 제미나이(Gemini)가 해당 장소의 '역사적 맥락'이나 '방문하기 가장 좋은 시간대'를 3줄 요약 제공.Photo Spot: "한국인들이 인스타그램 사진을 찍는 정확한 각도" 정보 포함.3.3. Travel Action Button$$Book a Tour$$: 해당 장소로 가는 프라이빗 투어나 입장권 구매 링크 (Klook 등).$$Copy Address for Taxi$$: 한국 택시 기사님에게 보여줄 한국어 주소 복사 기능.4. 데이터 파이프라인 (Data Pipeline)Source: 한국관광공사 TourAPI (다국어 서비스).Processing:API에서 받은 원문 데이터를 Gemini에게 전달.외국인이 흥미를 느낄만한 'Storytelling' 요소 추가.Storage: Firestore의 rankings 컬렉션 내 category: "place"로 저장.5. 단계별 수익화 계획Step 1: 투어 예약 링크 (어필리에이트).Step 2: eSIM 및 교통카드(T-money) 구매 링크 연결.Step 3: 지자체 협업 (특정 지역 여행지 홍보 세션 운영).💡 넥스트 엔진의 제언이 모델은 '비용은 낮추고 신뢰도는 높이는' 가장 현명한 MVP 방식입니다. 특히 한국인들 사이에서 유행하는 '로컬 감성' 여행지는 외국인들에게 "진짜 한국을 경험한다"는 강력한 마케팅 포인트가 됩니다.
